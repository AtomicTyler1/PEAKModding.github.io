Making a config for your mod is really simple!

## Code Example

```c# title="Plugin.cs"
using BepInEx;
using BepInEx.Configuration;

[BepInPlugin("com.yourname.yourmod", "Your Mod Name", "1.0.0")]
public class Plugin : BaseUnityPlugin
{
    public static ConfigEntry<float> speed;
    public static ConfigEntry<string> name;

    void Awake()
    {
        Logger.LogInfo("Plugin has loaded!");
        speed = Config.Bind("General", "Speed", 20f, "Speed of the player");
        name = Config.Bind("General", "Name", "Whatever you want", "Name of the player");
    }

    void Start()
    {
        Logger.LogInfo($"Speed: {speed.Value}") // This will log "Speed: 20.0"
        Logger.LogInfo($"Name: {name.Value}") // This will log "Name: Whatever you want"
    }
}
```

## How to make a config

:::note
This guide is assuming you have references to BepInEx and have a project setup.
:::

First off, you will need references to the following:
```c# title="Plugin.cs"
using BepInEx;
using BepInEx.Configuration;
```

Then to make a new config you can simply use this line in your BaseUnityPlugin:
```c# title="Plugin.cs"
public static ConfigEntry<float> config;
```

There are different types of ConfigEntry's you can do all sorts of types, here are some example:
```c# title="Plugin.cs"
public static ConfigEntry<float> config;
public static ConfigEntry<string> config2;
public static ConfigEntry<int> config3;
public static ConfigEntry<bool> config4;
```

Once you have your desired amount of configs, we can now move onto registering our configs.
Per config you want to use the name of our config, the use the `Config.Bind()` method. We will want to put this into our `Awake()` method.
For every config it will require the following:
- Heading/Section
- Name of the config
- Default Value
- Description

Here is a code example of registering configs

```c# title="Plugin.cs"
void Awake()
{
    config = Config.Bind("General", "Config", 100f, "Config Description for float");
    config2 = Config.Bind("General", "Config Number 2", "Default Value String", "Config Description for string");
    config3 = Config.Bind("General", "Config Number 3", 50, "Config Description for int");
    config4 = Config.Bind("General", "Config Number 4", false, "Config Description for bool");
}
```

To access your config value it is really simple! Just make sure the configs have been registered before using them. This could be by a simple null check.

```c# title="Plugin.cs"
config.Value;
config2.Value;
config3.Value;
config4.Value;
```

To see your config in-game you must first launch the game with the mod then it'll generate the files in your `BepInEx > Config > {your_mod_guid}` folder
